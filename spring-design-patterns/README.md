# 项目结构

```
spring-design-patterns                # spring 设计模式
  ├── design-creation                 # 创建型模式
  │   ├── 01-design-singleton         # 单例模式
  │   ├── 02-design-factory           # 工厂模式
  │   ├── 03-design-abstractFactory   # 抽象工厂模式
  │   ├── 04-design-prototype         # 原型模式
  │   └── 05-design-builder           # 建造者模式
  ├── design-structure                # 结构型模式
  │   ├── 01-design-proxy             # 代理模式
  │   ├── 02-design-adapter           # 适配器模式
  │   ├── 03-design-decorator         # 装饰者模式
  │   ├── 04-design-bridge            # 桥接模式
  │   ├── 05-design-facade            # 外观模式
  │   ├── 06-design-composite         # 组合模式
  │   └── 07-design-flyweight         # 享元模式
  ├── design-behaviour                # 行为型模式
  │   ├── 01-design-template          # 模板方法模式
  │   ├── 02-design-strategy          # 策略模式
  │   ├── 03-design-command           # 命令模式
  │   ├── 04-design-chai              # 责任链模式
  │   ├── 05-design-state             # 状态模式
  │   ├── 06-design-observe           # 观察者模式
  │   ├── 07-design-mediator          # 中介者模式
  │   ├── 08-design-iterator          # 迭代器模式
  │   ├── 09-design-visitor           # 访问者模式
  │   ├── 10-design-memento           # 备忘录模式
  └──-└── 11-design-interpreter       # 解释器模式
```
